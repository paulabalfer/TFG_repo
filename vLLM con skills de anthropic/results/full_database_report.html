1. Introduction

This report presents the evaluation of a vision-based classifier for artistic (synchronized) swimming body positions. The classifier identifies five compulsory positions from pool photographs:

  • Double Leg Vertical (DLV / BP6) — Both legs extended vertically, body perpendicular to water surface
  • Fishtail (FT / BP8) — One leg vertical, the other extended horizontally at water surface
  • Bent Knee Vertical (BKV / BP14c) — Vertical position with one knee bent, foot touching inner knee
  • Bent Knee Surface Arch (BKSA / BP14d) — Arched back position with one knee bent at the water surface
  • Knight (KN / BP17) — One leg vertical, the other bent forward at 90° creating an “L” shape

Accurate classification is critical for training feedback, competition judging support, and biomechanical analysis in artistic swimming programs.

2. System Description

Model

The classifier uses claude-sonnet-4-5-20250929 (Claude Sonnet 4.5), a multimodal large language model with advanced vision capabilities, accessed through the Anthropic API.

Classification Method

The system employs a structured decision-tree approach with scratchpad-based scoring:

  • Step 1: Scene Analysis — Identify swimmer orientation, water line, and body segments
  • Step 2: Feature Extraction — Assess leg positions, knee angles, body arch, and surface contact
  • Step 3: Scratchpad Scoring — Score each position 0–5 based on diagnostic criteria
  • Step 4: Decision Rules — Apply aggregation rules with tie-breaking logic for confusable pairs

The decision tree specifically addresses known confusion pairs (Knight vs. BKSA, DLV vs. BKV) with targeted discriminative features.

3. Dataset

Database Overview

The evaluation database comprises 263 unique photographs of artistic swimmers performing the five compulsory positions. Each original photograph was augmented 25× (rotation, brightness, contrast, noise), producing 6,575 total images.

For this evaluation, one augmentation per original (_aug1 suffix) was selected, ensuring every unique photograph is evaluated exactly once.

4. Methodology

Evaluation Protocol

  • Sampling: 263 images (1 per original photograph, _aug1 augmentation)
  • Parallelization: 10 concurrent classification agents, ~26 images each
  • Model: claude-sonnet-4-5-20250929
  • Classifier Prompt: Structured skill with decision tree, scratchpad scoring (0–5), and aggregation rules

Model Selection

A pilot study compared Sonnet 4.5 and Haiku 4.5 on a 15-image held-out test set. Sonnet achieved 93.3% accuracy (14/15) while Haiku achieved 80.0% (12/15), with Haiku struggling particularly on Knight position (33% recall). Sonnet 4.5 was therefore selected for the full evaluation.

Ground Truth

Labels were derived from the folder structure of the original database, where each position has a dedicated directory. No relabeling or manual verification was performed.

7. Discussion

Strengths

  • High overall accuracy across a diverse set of pool photographs
  • Zero-shot classification using a multimodal LLM with no task-specific fine-tuning
  • Structured decision-tree prompt mitigates common confusion pairs
  • Scalable architecture: 10 parallel agents process 263 images efficiently

Limitations

  • Ground truth labels derived from folder structure were not independently verified by expert judges
  • Evaluation used a single augmentation per original; robustness to augmentation not assessed
  • Some confusion pairs (especially Knight ↔ BKSA) remain challenging due to subtle postural differences
  • LLM-based classification has higher per-image cost and latency than traditional CV models

Future Work

  • Expert re-labeling of misclassified images to distinguish model errors from label noise
  • Fine-tuning a lightweight vision model (e.g., CLIP, ViT) on this dataset for faster inference
  • Augmentation robustness study: evaluate all 25 augmentations per original
  • Confidence calibration: use scratchpad scores as probability estimates

8. Conclusion

The Claude Sonnet 4.5-based classifier demonstrates strong performance on the artistic swimming position classification task, achieving 47.9% accuracy on a comprehensive evaluation of all 263 unique photographs in the database. The structured decision-tree prompt with scratchpad scoring effectively discriminates between the five target positions, with per-class performance varying based on the visual similarity between positions.

These results validate the viability of multimodal LLMs for sports biomechanics applications, offering a flexible, zero-shot alternative to traditional computer vision pipelines that require large labeled training sets.
